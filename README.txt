Grand Luxury Tree (Starter)
- Open index.html via a local server (recommended) so textures load correctly.
- Put your own photos into /IMG and name them 1.jpg ... 7.jpg (or update CONFIG.images in index.html).

Quick run (any one):
- VSCode Live Server
- Python:  python -m http.server 8000  then open http://localhost:8000/grand-luxury-tree/
